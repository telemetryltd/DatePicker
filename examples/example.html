Basic DatePicker

DatePicker in the most basic form.
Code:
$('#datepicker').DatePicker({
  mode:"single",
  date: new Date(),
  inline: true
});

Range DatePicker

Allows picking between two dates. Click a date to mark an endpoint, then another to set other end point.
Code:
$('#datepicker').DatePicker({
  mode:"range",
  date: new Date(),
  inline: true,
  calendars: 2
});

DatePicker with Min/Max

Demonstrates the new functionality of constraining the picker within boundaries.
Code:
$('#datepicker').DatePicker({
  mode:"single",
  date: new Date(),
  inline: true,
  minDate: "2012/10/24",
  maxDate: "2012/12/17"
});

DatePicker with Highlighting

This version of the DatePicker will highlight every day that is indicated by the periodicity selected below.
Repeats:
Code:
$('#datepicker').DatePicker({
  mode:"single",
  date: new Date(),
  inline: true,
  calendars: 2,
  onChange: function(el, date) {
    console.log("Changed to", el, date);
  }
});
